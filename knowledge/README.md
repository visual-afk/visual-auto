# 지식베이스 (AI의 뇌)

이 폴더의 파일들은 AI가 블로그 글을 쓸 때 참고하는 핵심 지식입니다.
파일을 수정하고 싶으면 Claude Code에게 말하세요.

## 뇌1: SEO 전략 — `seo/`
- `naver-seo-rules.md` — 네이버 블로그 SEO 규칙
- `keyword-master.md` — 서비스별 메인/롱테일 키워드
- `keyword-seasonal.md` — 시즌별 키워드
- `search-intent.md` — 키워드→검색의도 매핑 (AI 초안 가능)
- `competitor-gaps.md` — 경쟁사 분석, 틈새 주제

## 뇌2: 미용전문성 + 브랜딩 — `brand/`
- `salon-identity.md` — 살롱 소개, 위치, 분위기, 차별점
- `brand-voice.md` — 톤앤매너, 말투 규칙
- `services-menu.md` — 전 서비스 + 가격 + 과정 + 시간
- `staff-expertise.md` — 디자이너별 전문분야
- `product-lines.md` — 사용 제품, 선택 이유
- `case-studies.md` — 비포/애프터 사례, 고객 후기

## 뇌3: 소비자 심리 — `consumer/`
- `personas.md` — 타겟 고객 페르소나
- `pain-points.md` — 고객 불안/고민
- `desire-triggers.md` — 예약 유발 감정
- `decision-journey.md` — 검색→비교→예약 여정 (AI 초안 가능)
- `cta-patterns.md` — 예약 유도 문구 패턴 (AI 초안 가능)

## 채우는 방법

Claude Code에게 이렇게 말하면 됩니다:
- "브랜드 보이스 파일 채워줘" + 내용 설명
- "키워드 추가해줘" + 키워드 목록
- "고객 사례 추가해줘" + 사례 내용
- "CTA 패턴 초안 써줘" (AI가 먼저 써줌)

수정 후 "올려줘"라고 하면 Git에 반영됩니다.
