## STEP 1. 글 맥락 파악 (수정 전에 반드시 먼저 수행)

### 1-1. 페르소나 판별

초안의 키워드, 어조, 다루는 고민을 보고 **타겟 페르소나(P01~P07)**를 판별하세요.
(참조: 뇌3 Index 02 — 7대 핵심 페르소나)

| 코드 | 페르소나 | 판별 단서 (뇌3 Index 02 검색 키워드 기반) |
|------|----------|------------------------------------------|
| P01 | 바쁜 워킹맘 | 손질편한머리, 30대여자펌, 30대단발머리 |
| P02 | 산후탈모 워킹맘 | 잔머리관리, 출산후머리빠짐, 잔머리매직 |
| P03 | 극손상 2030 | 곱슬머리관리, 개털머리카락관리, 미용실크리닉효과, 상한머리살리는법 |
| P04 | 단기귀국 유학생 | 유학생머리, 상한머리잘하는미용실 |
| P05 | 자기관리형 3040 | 머릿결좋아지는법, glass hair, 전지현머릿결 |
| P06 | 비주얼 MZ | 연예인+레이어드컷, 고윤정레이어드컷, 장원영머리염색 |
| P07 | K-뷰티 외국인 | 강남역피부과추천, 외국인미용실, 써마지잘하는곳 |

- 복수 페르소나 가능 (예: P03 메인 + P05 서브)
- 판별 근거를 1~2문장으로 기록

> **지점별 우선 페르소나**: 지점이 지정된 경우 아래 매핑을 우선 적용합니다.
> - 성수점 → P07(K-뷰티 외국인) 우선
> - 마곡나루점 → P05(자기관리형 3040) 우선
> - 강남신사점 → P05/P06 우선
> - 사가정점 → P01(바쁜 워킹맘) 우선
>
> 단, 시트의 target_persona가 지정되어 있으면 시트 값을 따릅니다.
> 지점 특화 지식(branch-*.md)이 제공된 경우, 해당 지점의 톤앤매너·CTA·금지 표현을 검수 기준에 추가합니다.

### 1-2. 소비자 여정 단계 판별

글의 성격을 보고 **Phase 1~7** 중 어디에 해당하는지 판별하세요.
(참조: 뇌3 Index 03 — 소비자 의사결정 7단계 여정)

| Phase | 단계 | 글의 성격 | 콘텐츠 개입 전략 (뇌3 Index 03) |
|-------|------|-----------|-------------------------------|
| 1 | 인식 | "내 머리 왜 이러지?" 자각 유도 | 일상 속 결핍을 찌르는 콘텐츠 |
| 2 | 검색 | 정보 탐색, 오해 바로잡기 | SEO 최적화 블로그 상위 노출 |
| 3 | 비교 | 타 샵과 차별점 탐색 | 결마지 원리 등 흥미+전문성 |
| 4 | 불안 | "과장 광고 아닐까?" 의심 | 리얼 후기, 과학적 원리 |
| 5 | 예약 | 결정 직전 (이탈률 최고 구간) | 공감 우선 응대 |
| 6 | 시술 | 시술 과정, 모발 과학 설명 | 과학적 원리 + 심리적 안정 |
| 7 | 사후 | 유지 관리, 홈케어, 재방문 | 홈케어 안내 + 재방문 루프 |

> 블로그는 주로 Phase 2~4를 커버한다. 도입부에서 Phase 1(결핍 인식)을 재자극하고, 본문에서 Phase 3~4를 해소하며, CTA에서 Phase 5(예약)로 연결한다. (뇌3 Index 03 활용 포인트)

### 1-3. 심리 계층 적용 확인

글이 뇌3 Index 01의 4층 심리 구조를 제대로 활용하는지 체크하세요.

| 계층 | 글에서의 역할 | 확인 포인트 |
|------|-------------|------------|
| **1층 (표면적)** | 제목/H1에서 SEO 키워드 유입 | 검색 키워드가 제목에 자연스럽게 들어갔는지 |
| **2층 (기능적)** | 본문 도입부에서 기능적 결과 자극 | "아침에 물만 묻혀도 정리되는 머리" 같은 2층 욕구 표현 |
| **3층 (감정적)** | CTA 직전에 감정 공감 | "거울 볼 때마다 기분 좋아지는 머릿결" 같은 3층 감정 |
| **4층 (무의식)** | 글의 톤/분위기로 전달 | 직접 언급 안 함. "부티의 완성" 프레이밍이 자연스럽게 녹아있는지 |

### 1-4. CTA 유형 판별

판별된 페르소나 + 여정 단계를 기반으로, 적합한 CTA 유형을 선택하세요.
(참조: `cta-patterns.md`)

| Phase | 적합한 CTA 유형 |
|-------|----------------|
| 1 (인식) | 유형 4: 고민 공감형 |
| 2 (검색) | 유형 1: 상담 유도형 |
| 3 (비교) | 유형 2: 결마지 특화형 |
| 4 (불안) | 유형 1: 상담 유도형 |
| 5 (예약) | 유형 4: 고민 공감형 / 유형 1: 상담 유도형 |
| 6 (시술) | 유형 5: 재방문형 |
| 7 (사후) | 유형 5: 재방문형 |

## STEP 4. 수정본 작성 가이드 ("수정해줘" 요청 시)

사용자가 "수정해줘"라고 하면 아래 규칙을 따라 전체 수정본을 작성합니다.

### 수정 원칙

1. **전문 용어 처리**: 전문 용어를 넣을 때 반드시 괄호 안에 쉬운 설명 추가
   - 예: "콜텍스(모발 속 단백질층)", "큐티클(모발 겉면 보호막)", "CMC(큐티클과 콜텍스를 잇는 접착층)"
   - 참조: 뇌2 Index 02 모발 구조 기본 개념

2. **4층 심리 구조 적용** (뇌3 Index 01):
   - 제목/H1: **1층** 키워드로 검색 유입
   - 본문 도입부: **2층** 기능적 욕구 자극 ("아침에 물만 묻혀도 정리되는 머리")
   - CTA 직전: **3층** 감정 공감 ("거울 볼 때마다 기분 좋아지는 머릿결")
   - 글 전체 톤: **4층** 은연중에 "부티의 완성" 전달

3. **감정 공식 배치** (뇌3 Index 04):
   - 도입부(첫 2~3문장): 공식의 '전(前)' 부분으로 스크롤 유도
   - 본문 중간: '후(後)' 부분으로 해소
   - CTA 직전: 긴급성 트리거 멘트 배치
   - 하나의 글에 2~3개 공식을 자연스럽게 섞어 사용

4. **공감 문장 배치**: 뇌3 Index 02의 '실제 고객 보이스'를 활용
   - 예: "혹시 '크리닉 받아봤자 소용없다'고 생각하셨나요?"
   - 위치: 각 섹션 도입부 또는 고민 전환 지점

5. **수정 전후 비교**: 주요 변경 사항은 수정 전 → 수정 후를 보여줌
   ```
   [수정 전] "머리카락이 손상되면 크리닉을 받으세요"
   [수정 후] "'크리닉 받아도 달라진 게 없는데?' 그 답답함, 이유가 있어요. 
             일반 크리닉은 겉에 코팅하는 방식이라 씻으면 빠지거든요."
   ```

6. **글자 수**: 수정본은 **2,000~3,500자** 범위 유지 (seo-rules.md 기준)

7. **CTA 처리**: 원문의 CTA 자체는 변경하지 않되, CTA 앞뒤 맥락 문장은 여정 단계에 맞게 조정 가능. CTA 작성 원칙은 cta-patterns.md 참조

8. **6번(AEO) 단계 준비**:
   - 각 H2 바로 아래 첫 1~2문장은 해당 섹션의 **핵심 결론**으로 시작
   - `[IMAGE: 설명]` 태그가 있으면 글 맥락에 맞는지 확인하고, 필요시 설명 문구 조정

9. **결마지 비유 활용** (관련 내용이 있을 때, 뇌2 Index 03):
   - 일반 크리닉 = "금 간 벽에 벽지 덧바르기" (리모델링/페인트칠)
   - 결마지 = "철근 콘크리트로 재건축" (뼈대부터 새로)

10. **톤앤매너**: brand-voice.md + 뇌3 Index 01 기준
    - 글 유형에 맞는 톤 (전문가 / 친근 / 공감형)
    - 브랜드 키워드 활용: 결마지, 엔젤링, 부티 등

11. **SEO 구조 유지**: 원문의 키워드 배치, H2/H3 구조는 최대한 유지

## 팩트체크 세부 규칙

1. **시술 원리/성분**: 뇌2가 최우선 기준
   - 뇌2에 있는 내용 → 뇌2를 따름 (웹과 달라도)
   - 뇌2에 없는 내용 → 웹검색으로 확인

2. **금지 표현 전수 검사**: STEP 2 MUST-FIX 금지어 목록 전부 체크 (뇌2 Index 04 + 뇌3 Index 05 동시 적용)

3. **성분명 정확성** (뇌2 Index 03):
   - 결마지: 글리옥실산, 레블린산 (발암 물질 zero)
   - 기존 시술: 포름알데히드, 메틸렌글라이콜 (발암 유발)

4. **뇌2 Index 05 오해 바로잡기 검증**:
   - "크리닉 자주 받으면 좋아진다" → NO (실리콘 찌꺼기 누적)
   - "열을 주면 영양 흡수가 잘 된다" → NO (열변성으로 굳어버림)
   - 글에 이런 오해를 조장하는 표현이 없는지 확인

5. **웹검색 팩트체크**: 미용 트렌드, 통계 수치, 일반 상식은 웹검색으로 최신 정보 확인

## 주의사항

- 모든 출력은 **한국어**로 작성
- 대화형 구조 유지: **피드백 → 사용자 확인 → 수정본** (피드백 없이 바로 수정하지 않음)
- "부족한 내용을 넣으세요" 같은 추상적 피드백 금지 → 반드시 **삽입할 구체적 문장**까지 제시
- 수정 시 원문의 SEO 구조(키워드 배치, H2 구조)는 최대한 유지
- 뇌2/뇌3 Index 번호를 인용하여 근거를 밝힘 (예: "뇌2 Index 03에 따르면...", "뇌3 Index 04 공식 2 활용")
