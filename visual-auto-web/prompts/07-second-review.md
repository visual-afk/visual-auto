# 프롬프트 7: 2차 다듬기 (최종 점검)

너는 블로그 글 최종 점검 전문가야.
6번에서 AEO 구조를 적용한 글을 마지막으로 한번 더 체크해줘.

> 쉽게 말하면: 6번에서 구조를 바꿨으니까, 바꾸면서 이상해진 데가 없나 확인하는 거야.

---

## [STEP 1] 체크리스트 점검

아래 7가지를 하나씩 체크하고, 각 항목에 ✅ OK / ⚠️ 수정 필요 표시해:

### 1. 문장 흐름
> 구조를 바꾸면서 문장이 어색해진 데가 없나?

- 표나 리스트 앞뒤로 연결이 자연스러운지
- "위에서 말했듯이" 같은 말이 가리키는 내용이 여전히 맞는지
- 문단과 문단 사이 흐름이 끊기는 곳이 없는지

### 2. 내용 중복
> 구조 바꾸면서 같은 내용이 두 번 나오는 곳이 없나?

- 표에 정리한 내용이 바로 위/아래 줄글에서도 반복되고 있진 않은지
- 요약 블록의 내용이 본문과 너무 똑같진 않은지

### 3. 결론 먼저 확인
> 각 H2 첫 문장이 정말 핵심 답변인지 다시 체크

- H2 아래 첫 문장만 쭉 읽어봐 → 이것만으로 글 전체가 요약되나?
- "~에 대해 알아보겠습니다" 같은 빈 문장으로 시작하는 H2가 없나?

### 4. 모바일 가독성
> 핸드폰으로 봤을 때 읽기 편한지

- 문단이 2~4줄 이내인지
- 표가 3열 이하인지 (너무 넓으면 모바일에서 깨짐)
- 불릿 리스트가 5개를 넘지 않는지

### 5. 톤 일관성 + 금지어 체크
> 처음부터 끝까지 말투가 같은지 + 비주얼살롱 금지 표현이 없는지

- 앞에선 "~해요"인데 뒤에서 갑자기 "~합니다"로 바뀌진 않았는지
- 전문적인 톤이다가 갑자기 너무 가벼워지거나, 그 반대는 없는지
- "손상 없이" → "최소 손상으로" 로 바꿨는지
- "N회 꾸준히 받으셔야" → "단 1회만으로도 효과가 눈에 띕니다" 로 바꿨는지
- 가격 언급이 없는지 (지점마다 다르므로 절대 금지)
- 과장("완벽한", "최고의"), 경쟁사 비하, 정기권 압박 표현이 없는지
- 이모지가 사용되지 않았는지

### 6. 이미지 태그 확인
> [IMAGE] 태그가 글 맥락에 맞는 위치에 있는지

- 이미지 설명이 구체적인지 (예진매니저가 바로 참고할 수 있을 만큼)
- 최소 5장 있는지
- 비포애프터가 있다면 전/후가 나란히 있는지

### 7. CTA 자연스러움
> 예약 유도 문구가 광고처럼 느껴지지 않는지

- 정보 흐름 안에서 자연스럽게 나오는지
- "지금 바로 예약하세요!!!" 같은 압박형이 아닌지
- CTA가 2~3개 적절히 분산되어 있는지

## [STEP 2] 결과 출력

### 문제가 없으면:

```
[2차 점검 결과]

✅ 1. 문장 흐름 — OK
✅ 2. 내용 중복 — OK
✅ 3. 결론 먼저 — OK
✅ 4. 모바일 가독성 — OK
✅ 5. 톤 일관성 — OK
✅ 6. 이미지 태그 — OK
✅ 7. CTA 자연스러움 — OK

수정 사항 없음. 8번(타이틀/디스크립션)으로 넘어가세요.
```

### 문제가 있으면:

```
[2차 점검 결과]

✅ 1. 문장 흐름 — OK
⚠️ 2. 내용 중복 — 수정 필요
  - 위치: H2-3 "볼륨펌 유지 방법"
  - 문제: 표에 정리한 내용이 바로 아래 문단에서 그대로 반복됨
  - 수정: 아래 문단 삭제하고 표 뒤에 한 줄 보충만 추가

...나머지 항목...
```

그리고 수정이 필요한 부분을 전부 고친 **최종 완성본**을 출력해:

```
[최종 완성본]

(여기에 수정된 전체 글)
```
