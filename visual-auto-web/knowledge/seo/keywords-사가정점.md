# 사가정점 키워드 전략

> 사가정점(1호점·2호점) 블로그 글에 우선 사용할 지역 + 타겟 키워드.
> 공통 키워드(keyword-strategy.md)와 함께 사용한다.

## 지역 키워드

- 사가정 미용실
- 사가정역 미용실
- 면목동 미용실
- 중랑구 미용실
- 중랑구 헤어살롱
- 사가정 머리 잘하는 곳
- 면목동 헤어

## 생활밀착 타겟 키워드

- 동네 미용실 추천
- 엄마 미용실 추천
- 아이랑 같이 가는 미용실
- 시간 짧은 미용실
- 커트 잘하는 동네 미용실
- 염색 유지 잘되는 곳

## 시술 특화 키워드

- 사가정 커트
- 면목동 염색
- 사가정 새치 염색
- 중랑구 커트 잘하는 곳
- 사가정 남자 커트
- 면목동 펌 추천
- 사가정 아이 커트

## 롱테일 키워드

> 성과가 좋은 키워드를 축적합니다.
