# 비주얼살롱(본사) 키워드 전략

> 비주얼살롱 공식 계정 글에 우선 반영할 키워드. 성과가 확인되면 예진매니저가 계속 보강한다.

## 브랜드 키워드

- 비주얼살롱, VISUAL SALON, 비주얼살롱 지점

## 시술 키워드

- 결마지, 결까지 관리하는 매직
- 손상모 케어, 머릿결 관리, 곱슬머리 매직
- 산성 스트레이트, 헤어 클리닉

## 타겟 키워드

- 머릿결 좋아지는 미용실
- 손상모 잘하는 미용실
- 곱슬머리 잘하는 미용실
- 결마지 잘하는 곳

## 롱테일 키워드

> 성과가 좋은 키워드를 축적합니다.

- 10년 곱슬 결마지 후기
- 매직이랑 결마지 차이
