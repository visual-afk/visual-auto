# 네이버 블로그 SEO 규칙

## 글 구조

- 최적 길이: 2,000~3,500자 (네이버 상위 노출 기준)
- 이미지 수: 5~10장 (텍스트 대비 적절한 비율 유지)
- 소제목 사용 규칙: H2는 3~6개, H3는 필요한 경우에만
- 첫 문단에 핵심 키워드 자연스럽게 포함
- 결론/요약을 먼저 제시하는 AEO 구조 적용

## 네이버 C-Rank 최적화

- 하나의 주제에 집중하는 전문성 있는 글 작성
- 미용/헤어 카테고리에서 일관된 콘텐츠 발행
- 주 2~3회 꾸준한 발행 유지
- 독자와의 소통 (댓글 답변, 공감)
- 표절/복사 콘텐츠 절대 금지

## DIA (Deep Intent Analysis) 대응

- 검색 의도에 정확히 부합하는 콘텐츠 작성
- 문서의 체류 시간을 높이는 구성 (목차, 이미지, 단계별 설명)
- 클릭 후 이탈률을 낮추는 충분한 정보 제공

## 태그 전략

- 메인 키워드 + 서비스 키워드 + 지역 키워드 조합
- 태그 5~10개 사용
- 예시: 강남미용실, 비주얼살롱, 볼륨펌, 강남펌추천, 헤어트렌드

## 제목 패턴

### 잘 되는 제목 예시

- [키워드] + 구체적 정보: "볼륨펌 종류 3가지, 얼굴형별 추천"
- [키워드] + 경험/후기: "레이어드컷 실제 시술 후기, 유지 기간은?"
- [키워드] + 비교/선택: "히피펌 vs 바디펌, 어떤 게 나한테 맞을까"

### 피해야 할 제목

- 키워드 나열: "강남미용실 펌 염색 커트 추천"
- 과장 표현: "인생 최고의 펌! 무조건 여기!!"
- 너무 긴 제목: 30자 초과

## AEO/GEO 구조 규칙

- AI 답변 활용을 고려한 구조화된 콘텐츠
- 질문형 소제목 활용 (예: "볼륨펌 유지 기간은?")
- 핵심 내용은 문단 첫 줄에 결론 먼저 제시
- 비교가 필요한 경우 표 형태로 정리
- 방법/절차는 번호 리스트로 구성

## 모바일 가독성 규칙

- 한 문단 2~4줄 이내
- 짧고 명확한 문장
- 이모지 사용하지 않음
- 인용구/콜아웃은 제한적 사용
- 이미지는 섹션 전환점에 배치
