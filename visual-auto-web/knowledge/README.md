# Knowledge Base (SSOT 지식베이스)

이 폴더의 마크다운 파일들은 AI가 블로그 글을 쓸 때 참고하는 핵심 지식입니다.
예진매니저가 직접 작성하고 관리합니다.

## 구조

### seo/ — SEO 전략
- `seo-rules.md` — 네이버 블로그 SEO 규칙, 최적 글 길이, 태그 전략
- `keyword-strategy.md` — 타겟 키워드 목록, 시즌별 키워드

### brand/ — 브랜드 & 전문성
- `brand-voice.md` — 비주얼살롱 톤앤매너, 말투 가이드
- `services.md` — 서비스 메뉴, 가격, 시술 과정
- `expertise.md` — 미용 전문 지식, 기법, 트렌드

### consumer/ — 소비자 심리
- `psychology.md` — 고객 의사결정 패턴, 불안요소, 욕구 트리거
- `personas.md` — 타겟 고객 페르소나

## 작성 가이드

- 마크다운으로 작성 (제목은 ##, 목록은 -)
- 구체적이고 실제적인 내용 (추상적 X)
- 비주얼살롱의 실제 경험과 노하우 반영
- 주기적으로 업데이트 (새 서비스, 트렌드 변화 등)
