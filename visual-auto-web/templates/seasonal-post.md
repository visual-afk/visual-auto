# 시즌형 블로그 템플릿

계절/트렌드에 맞는 시의성 있는 글 구조.

## 구조

1. **시즌 이슈** (2-3문장)
   - 이번 시즌의 트렌드/이슈
   - 왜 지금 이 주제가 중요한지

2. **트렌드 분석** (2-3 섹션)
   - 올해 유행하는 스타일/관리법
   - [IMAGE: 트렌드 예시]
   - 비주얼살롱의 해석

3. **실용적 팁**
   - 계절에 맞는 헤어 관리법
   - 집에서 할 수 있는 것 vs 살롱에서 해야 하는 것

4. **추천 시술**
   - 이번 시즌에 인기 있는 시술
   - [IMAGE: 시술 결과]
   - 하드 CTA (시즌 한정 프로모션이 있다면)
