# 「리뷰 답글」 기능 강의 대본 (디자이너용 · 3~5분)

> 예진매니저가 화면 녹화하며 그대로 읽는 **영상 대본**이에요.
> 각 장면 = **[화면] 무엇을 보여주고 / [대사] 뭐라고 말하고 / [자막] 화면에 얹을 캡션**.
> 대사는 편하게 읽되, **버튼 이름은 화면에 실제로 뜬 그대로** 눌러 주세요.
>
> ⏱ 목표 길이: **약 4분** · 대상: **디자이너(매장에서 리뷰 답글 다는 사람)**

---

## 🎬 S0. 오프닝 (0:00 – 0:20)

**[화면]** 스마트플레이스에 고객 리뷰가 쭉 달려 있는 모습 → 앱 「리뷰 답글」 화면으로 전환

**[대사]**
> "안녕하세요! 오늘은 고객님이 남긴 리뷰에, **매장 톤 답글을 30초 만에** 다는 방법을 알려드릴게요.
> 이제 뭐라고 답할지 고민 안 하셔도 돼요. 붙여넣고, 버튼 한 번이면 끝이에요. 같이 해볼까요?"

**[자막]** `리뷰 답글, 30초면 끝 ✍️`

---

## 🎬 S1. 들어가기 (0:20 – 0:40)

**[화면]** 로그인된 상태에서 왼쪽(또는 아래) 메뉴의 **「리뷰 답글」** 클릭

**[대사]**
> "앱에 로그인하면, 메뉴에서 **「리뷰 답글」** 을 눌러요.
> _(본사 계정만 해당) 여러 지점을 관리하시면 위에서 '어느 지점 리뷰예요?' 를 먼저 골라주세요. 디자이너분들은 우리 지점이 자동으로 잡히니까 그냥 넘어가면 돼요._"

**[자막]** `메뉴 → 「리뷰 답글」`

---

## 🎬 S2. 리뷰 가져오기 — 기본은 '복붙' (0:40 – 1:40)

**[화면]** **「이 지점 리뷰 보러가기」** 클릭 → 네이버 리뷰 페이지 새 탭 → 리뷰 하나 드래그해서 복사 → 앱으로 돌아와 **「고객 리뷰 붙여넣기」** 칸에 붙여넣기

**[대사]**
> "먼저 답글 달 리뷰를 가져올게요.
> **「이 지점 리뷰 보러가기」** 를 누르면 우리 매장 네이버 리뷰 페이지가 새 창으로 열려요.
> 여기서 답글 달고 싶은 리뷰를 **드래그해서 복사**하고요,
> 다시 앱으로 돌아와서 **「고객 리뷰 붙여넣기」** 칸에 **붙여넣기** 하면 돼요.
>
> 참고로, 리뷰를 자동으로 쫙 긁어오는 기능도 있는데 **지금은 네이버가 막아둬서 들쭉날쭉해요.**
> 그러니까 **가장 확실한 건 이렇게 복사·붙여넣기** 예요. 이게 정답이에요!"

**[자막]** `리뷰 복사 → 「고객 리뷰 붙여넣기」에 붙여넣기` / `제일 확실한 방법 = 복붙 👍`

---

## 🎬 S3. 시술 고르고 답글 만들기 (1:40 – 2:20)

**[화면]** **"어떤 시술 리뷰예요? (선택)"** 에서 시술 칩(예: 결마지, 펌) 탭 → **「답글 써드릴게요」** 클릭 → 로딩

**[대사]**
> "붙여넣었으면, 아래에서 **어떤 시술 리뷰인지** 골라줄 수 있어요. 결마지, 펌, 염색, 클리닉, 컷 중에서요.
> 이건 **선택**이에요. 고르면 답글이 그 시술에 더 딱 맞게 나와요.
> 자, 이제 **「답글 써드릴게요」** 를 누르면… (잠깐 기다리면) 답글이 뚝딱 만들어져요!"

**[자막]** `시술 선택(선택사항) → 「답글 써드릴게요」`

---

## 🎬 S4. 답글 고르고 발행하기 (2:20 – 3:20)

**[화면]** "AI 답글 · 매장 톤" 답글 2~3개가 뜬 모습 → **"상위노출 키워드 반영"** 줄을 손가락/커서로 가리킴 → 맘에 드는 답글의 **「복사하고 스마트플레이스 열기」** 클릭 → 스마트플레이스에서 붙여넣기 발행

**[대사]**
> "이렇게 **매장 톤 답글이 2~3개** 나왔어요. 마음에 드는 걸로 고르면 돼요.
> 여기 밑에 **'상위노출 키워드 반영'** 이라고 보이시죠? 우리 매장이 노출되고 싶은 키워드를 답글에 슬쩍 녹여준 거예요. 검색에 도움이 돼요.
> 맘에 드는 답글에서 **「복사하고 스마트플레이스 열기」** 를 누르면, 답글이 자동으로 복사되고 스마트플레이스가 열려요.
> 거기서 **붙여넣기 하고 등록**하면 끝!
> 살짝 아쉬우면 **「다시 쓰기」** 로 다른 버전을 받을 수도 있어요.
> 아, 그리고 이 답글은 **따로 저장되지 않으니까**, 마음에 들면 **그 자리에서 바로 발행**해 주세요."

**[자막]** `답글 선택 → 「복사하고 스마트플레이스 열기」 → 붙여넣기 발행` / `저장 안 됨 = 그 자리에서 바로 발행 ⚠️`

---

## 🎬 S5. (보너스) 리뷰가 많을 때 — 자동 가져오기 (3:20 – 4:30)

**[화면]** **「리뷰 자동 가져오기」** 클릭 → 설정 모달 → "리뷰 가져오기" 버튼을 북마크바로 드래그 → 네이버 리뷰 페이지에서 스크롤 → 북마크 클릭 → 앱에 "가져온 리뷰" 목록이 뜨고, 하나 눌러서 답글쓰기

**[대사]**
> "리뷰가 한두 개면 복붙이 제일 빠른데요, **한 번에 여러 개** 답글 달아야 할 땐 이 방법도 있어요.
> **컴퓨터(크롬)** 에서만 되는데요, **「리뷰 자동 가져오기」** 를 누르면 설정 창이 떠요.
> 여기 **'리뷰 가져오기'** 버튼을 **북마크바로 딱 한 번 끌어다 놓으면** 설치 끝이에요. _(북마크바는 Ctrl 또는 ⌘ + Shift + B로 켜요.)_
> 그다음엔, 네이버 리뷰 페이지에서 리뷰가 보이게 **쭉 스크롤**하고, 방금 만든 **'리뷰 가져오기' 북마크를 클릭**하면,
> 우리 앱으로 **리뷰가 한꺼번에 쏙** 들어와요. 그중 하나를 누르면 바로 답글쓰기로 이어져요.
> 혹시 이게 안 되면? **미련 없이 복붙하세요!** 그게 항상 되니까요. 📱 휴대폰은 복붙만 돼요."

**[자막]** `많을 때: 「리뷰 자동 가져오기」(PC 크롬)` / `안 되면? 그냥 복붙 😉`

---

## 🎬 S6. 마무리 (4:30 – 5:00)

**[화면]** 완성된 답글이 스마트플레이스에 달린 모습 → 앱 「리뷰 답글」 화면

**[대사]**
> "정리할게요. ①리뷰 **복사해서 붙여넣기**, ②시술 고르고 **「답글 써드릴게요」**, ③맘에 드는 답글 **복사해서 스마트플레이스에 발행**.
> 이 세 단계만 기억하시면 돼요.
> 리뷰는 몰아서 하루 한 번씩 답글 달아주면 딱 좋아요.
> 하다가 막히는 거 있으면 저(예진매니저)한테 편하게 물어보세요. 오늘도 고생 많으셨어요!"

**[자막]** `① 붙여넣기 → ② 답글 써드릴게요 → ③ 복사·발행` / `리뷰 답글 완성! 🎉`

## 🎬 촬영 준비물 · 셋업 체크

- [ ] **데모 계정**으로 로그인 (실제 고객정보 노출 주의 — 화면에 이름·번호 **워터마크**가 깔리니 편집 시 가려도 OK)
- [ ] 그 지점에 **네이버 리뷰 링크가 등록**돼 있어야 「이 지점 리뷰 보러가기」·「리뷰 자동 가져오기」 버튼이 보임 (없으면 스마트플레이스로 직접 이동하는 장면으로 대체)
- [ ] **예시 리뷰 텍스트 1개** 미리 준비 (복붙 시연용)
- [ ] S5 북마클릿 시연은 **PC 크롬**에서, 북마크바 미리 켜두기 (⌘/Ctrl+Shift+B)
- [ ] 스마트플레이스에 **미리 로그인** 해두면 발행 장면이 매끄러움

## ⚠️ 촬영 시 주의 (대본에 넣지 말 것)

- 예전 **서버 "리뷰 불러오기"** 버튼은 지금 UI에 **없어요.** 헷갈리지 않게 화면에서도 언급 X.
- 자동수집 = **「리뷰 자동 가져오기」(북마클릿)** 하나뿐. 이것도 안 될 수 있으니 **복붙을 메인**으로.
