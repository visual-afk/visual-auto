# 인스타그램 연동 설정 (본사/관리자용)

디자이너가 앱에서 "인스타그램 연결"을 누르면 본인 계정으로 로그인 승인 1회 → 이후 릴스 조회수·저장수가 자동으로 채워집니다.
이를 위해 본사에서 Meta(페이스북) 개발자 앱을 한 번 만들어야 합니다.

## 1. Meta 개발자 앱 만들기

1. https://developers.facebook.com 접속 → 페이스북 계정으로 로그인 → 「내 앱」 → 「앱 만들기」
2. 사용 사례에서 **"Instagram API with Instagram Login"** 선택 (페이스북 페이지 연결이 필요 없는 방식)
3. 앱 이름: `비주얼살롱` 등 아무거나

## 2. 설정값 입력

앱 대시보드 → Instagram → API setup with Instagram login 에서:

1. **리디렉션 URI 등록**: `https://<우리 앱 주소>/api/instagram/callback`
   (예: `https://visual-auto.vercel.app/api/instagram/callback` — 정확히 일치해야 함)
2. **앱 ID / 앱 시크릿** 복사 → Vercel 환경변수로 등록:

| 환경변수 | 값 |
|---|---|
| `INSTAGRAM_APP_ID` | Instagram 앱 ID |
| `INSTAGRAM_APP_SECRET` | Instagram 앱 시크릿 |
| `INSTAGRAM_REDIRECT_URI` | (선택) 위에서 등록한 리디렉션 URI. 안 넣으면 접속 주소로 자동 조립 |

3. 권한(Permissions): `instagram_business_basic`, `instagram_business_manage_insights`

## 3. 디자이너 계정 조건

- 인스타 계정이 **프로페셔널 계정(비즈니스 또는 크리에이터)** 이어야 합니다.
  - 개인 계정은 연동 불가 → 인스타 앱에서 설정 → 계정 유형 → 프로페셔널 계정으로 전환 (무료)
- 앱이 **개발 모드**인 동안은 Meta 앱에 "Instagram 테스터"로 추가된 계정만 연결됩니다.
  - 앱 대시보드 → App roles → Roles → Instagram Testers 에 디자이너 인스타 아이디 추가
  - 디자이너는 인스타 앱 → 설정 → 웹사이트 권한(Apps and Websites) → 테스터 초대에서 수락
- 전 지점 디자이너가 심사 없이 쓰려면 **App Review(고급 액세스)** 를 신청해 승인받아야 합니다.
  - 위 권한 2개에 대해 사용 목적(본인 콘텐츠 성과 확인)과 시연 영상을 제출

## 4. 동작 방식

- 디자이너: 「내 글·조회수」 화면 → 인스타그램 연결 → 인스타 로그인 승인
- 릴스에 발행 링크(인스타 릴스 주소)를 등록해두면, 하루 2회 화면 진입 시 자동으로 + "지금 수집" 버튼으로 수동으로 조회수·저장수를 가져옵니다
- 토큰은 60일짜리이며 만료 전에 자동 연장됩니다. 60일 넘게 앱을 안 쓰면 연결이 풀릴 수 있어요 → 다시 「연결하기」 한 번이면 됩니다

## 문제 해결

| 증상 | 원인/해결 |
|---|---|
| 연결하기를 눌러도 "연동 설정이 아직 안 됐어요" | 환경변수(`INSTAGRAM_APP_ID/SECRET`) 미설정 |
| 인스타 로그인 후 "연결에 실패했어요" | 개인 계정(프로페셔널 전환 필요) 또는 테스터 미등록(개발 모드) |
| 조회수가 안 들어와요 | 릴스에 인스타 링크가 등록됐는지, 링크가 본인 계정 게시물인지 확인 |
