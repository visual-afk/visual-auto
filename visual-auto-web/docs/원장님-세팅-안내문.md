# 원장님 세팅 & 사용 안내 (예진매니저용 가이드)

> 이 문서는 **예진매니저가 각 지점 원장님을 세팅하고, 원장님·디자이너가 앱을 쓰기 시작하도록** 돕기 위한 실무 가이드예요.
> 위→아래 순서대로 따라 하면 한 지점이 완성돼요.
>
> _기준일: 2026년 6월 30일 업데이트 반영_

---

## 🗺 전체 그림 (한 지점이 돌아가려면)

```
[본사(예진매니저)]  ─①─▶  지점 만들기
                    ─②─▶  원장님 계정 만들기 (휴대폰=아이디)
                    ─③─▶  원장님께 아이디·비번 + 사용법 전달
                              │
                              ▼
[원장님]            ─④─▶  우리 지점 디자이너/인턴 등록
                    ─⑤─▶  디자이너에게 아이디·비번 전달
                              │
                              ▼
[디자이너]          ─⑥─▶  로그인 → 내 네이버 아이디 등록 → 글쓰기 시작
```

한 지점당 **①~③은 예진매니저가**, **④~⑤는 원장님이**, **⑥은 디자이너가** 해요.

---

## ① 지점 만들기  _(예진매니저 · 본사 계정)_

**메뉴: 「지점 관리」 → 「지점 추가」**

지점마다 아래를 채워요.

| 항목 | 넣는 값 | 메모 |
|------|---------|------|
| **지점 이름** *(필수)* | 예: 잠실점 | |
| **지역** | 예: 서울 송파구 | |
| **지식베이스 slug** | 예: 잠실점 | 글 품질용 지식 파일 이름. 헷갈리면 지점 이름과 똑같이 |
| **아임웹 글쓰기 링크 (지점 공용)** | 그 지점 아임웹 글쓰기 화면 주소 | 디자이너가 「아임웹 열기」 누르면 이 주소가 열려요 |
| **출근체크 위치 (GPS)** | 위도·경도·허용 반경(m) | 아래 설명 참고 |

**GPS 좌표 넣는 법 (제일 편한 방법):**
- 그 **지점 매장 안에서** 원장님 휴대폰으로 앱을 열고 「지점 관리」 → **「📍 현재 위치로 설정」** 을 누르면 좌표가 자동으로 들어가요.
- **허용 반경**은 기본 **200m**. 매장이 크거나 주차장 포함이면 300m로 늘려도 돼요.
- 디자이너는 **이 반경 안에서만 출근**이 눌려요.

> 💡 **네이버 아이디는 여기서 안 넣어요.** 네이버는 **디자이너가 각자** 자기 블로그를 등록해요(⑥ 단계). 지점에는 **아임웹(공용)만** 넣어요.
>
> 💡 지점을 만들어도 **지식베이스 글은 자동으로 안 생겨요.** 글 품질을 위해 예진매니저가 따로 채워야 해요.

---

## ② 원장님 계정 만들기  _(예진매니저 · 본사 계정)_

**메뉴: 「지점·사람」 → 「바로 추가」**

1. **이름** 입력 (예: 김원장)
2. **휴대폰** 입력 — ⚠️ **이 번호가 곧 로그인 아이디**예요. 숫자만(예: 01012345678).
3. **임시 비번** — 기본값이 **`visual1234`** 로 들어가 있어요. 그대로 써도 되고 바꿔도 돼요.
4. **역할**을 **「원장」** 으로 선택
5. **지점**을 방금 만든 지점으로 선택
6. **「바로 추가」** 클릭

→ "계정이 만들어졌어요"와 함께 **아이디·비번**이 떠요. **「아이디·비번 복사」** 를 눌러요.

> 💡 **초대 링크 방식**도 있어요(「지점·사람」에서 초대 → 카톡 전송 → 본인이 비번 직접 설정). 하지만 원장님은 **「바로 추가」가 더 빠르고 확실**해요. 만든 아이디·비번을 바로 전달하면 끝이에요.

---

## ③ 원장님께 전달하기  _(예진매니저)_

복사한 내용을 카톡·문자로 보내요. 아래 문구를 그대로 쓰셔도 돼요. 👇

```
[비주얼살롱 원장님 앱 안내]

원장님, 지점 관리 앱 계정 만들어드렸어요 🙂

▸ 접속 주소: (앱 주소)
▸ 아이디: 010xxxxxxxx  (원장님 휴대폰 번호예요)
▸ 임시 비번: visual1234

처음 하실 일 3가지예요.
1) 로그인 → 「우리 지점 사람」에서 우리 디자이너·인턴을 등록해 주세요.
2) 「성과 대시보드」에서 우리 지점 매출·유입을 보실 수 있어요.
3) 아임웹은 앱에서 「아임웹 열기」 누른 화면에 한 번만 로그인해두면 계속 유지돼요.

궁금한 건 저(예진매니저)에게 편하게 물어보세요!
```

> 💡 원장님이 로그인 후 비번을 바꾸고 싶어 하면, 지금은 **본사가 「지점·사람」에서 다시 만들어(바로 추가) 새 비번을 전달**하는 방식이에요. (임시 비번 `visual1234`를 계속 써도 로그인엔 문제없어요.)

---

## ④ 원장님이 디자이너 등록하기  _(원장님 · 원장 계정)_

원장님께 아래 방법을 안내해 주세요. **두 가지 방법** 중 편한 걸 쓰면 돼요.

**메뉴: 「우리 지점 사람」**

### 방법 A — 바로 추가 (추천, 제일 빠름)
1. **「바로 추가」** 영역에서 이름 · 휴대폰 · 역할(디자이너 또는 인턴) 입력
2. 임시 비번은 **`visual1234`** 그대로 두기
3. **「바로 추가」** 클릭 → 뜨는 **아이디·비번을 복사**해서 그 디자이너에게 전달

### 방법 B — 초대 링크
1. **「새로 초대하기」** 에서 이름·연락처·역할 입력
2. **초대 링크**가 만들어지면 **「초대 링크 보내기」** 로 카톡·문자 전송
3. 디자이너가 링크 열어 **본인이 직접 비번 설정** 후 가입

> 💡 원장님은 **우리 지점 디자이너·인턴만** 등록/관리할 수 있어요. 다른 지점은 안 보여요. (원장 추가·본사 권한은 예진매니저만 가능)

---

## ⑤ 디자이너에게 전달하기  _(원장님 → 디자이너)_

원장님이 디자이너에게 보낼 문구 예시예요. 👇

```
[비주얼살롱 앱 안내]

우리 지점 블로그·릴스·출근 앱이에요 🙂

▸ 접속 주소: (앱 주소)
▸ 아이디: 010xxxxxxxx  (본인 휴대폰 번호)
▸ 임시 비번: visual1234

처음 들어가면 「글쓰기」에서 내 네이버 아이디를 한 번 등록해줘요.
출근은 매장 안에서 「출근」 눌러서 (그루밍 체크 + 사진) 하면 돼요.
```

---

## ⑥ 디자이너 첫 세팅  _(디자이너 · 본인 계정)_

디자이너가 처음 로그인하면 아래를 한 번씩 해두면 준비 끝이에요.

1. **로그인** (휴대폰 번호 + `visual1234`)
2. **내 네이버 아이디 등록** — 「글쓰기」 화면에서 `blog.naver.com/` 뒤에 **내 네이버 아이디**만 넣어요.
   - 예: 아이디가 `dearmy3rdtest` 면 그 부분만 입력 (전체 주소 X)
   - 한 번 등록 + 브라우저에 네이버 로그인 해두면, 다음부터 「네이버 블로그 열기」가 바로 내 글쓰기 창을 열어요.
3. **아임웹**은 지점 공용이라 따로 등록 안 해요. 「아임웹 열기」 누른 창에 **한 번 로그인**만 해두면 유지돼요.
4. 이제 **글쓰기 · 릴스 · 리뷰 답글 · 출근**을 쓰면 돼요. (자세한 사용법은 [사용설명서.md](사용설명서.md) 참고)

## ✅ 지점별 세팅 체크리스트 (예진매니저용)

지점 하나 열 때마다 이 표를 채우면 빠뜨림이 없어요.

| 항목 | 담당 | 완료 |
|------|------|:----:|
| 지점 생성 (이름·지역·slug) | 예진매니저 | ☐ |
| 아임웹 공용 글쓰기 링크 등록 | 예진매니저 | ☐ |
| 출근 GPS 좌표·반경 설정 | 예진매니저(현장) | ☐ |
| 지식베이스 글 채우기 | 예진매니저 | ☐ |
| 원장 계정 생성 (휴대폰=아이디) | 예진매니저 | ☐ |
| 원장께 아이디·비번 + 안내문 전달 | 예진매니저 | ☐ |
| 원장이 디자이너·인턴 등록 | 원장 | ☐ |
| 디자이너 네이버 블로그 등록 | 디자이너 | ☐ |
| (선택) HandSOS 성과 연동 | 예진매니저→개발 | ☐ |

> 💡 **HandSOS 성과 연동**: 「성과 대시보드」에 그 지점이 **"연동 전"** 으로 뜨면, 아직 매출 데이터가 안 붙은 거예요. 이건 앱 화면에서 못 바꾸니 **개발쪽(또는 본사 세팅)에 연동 요청**하면 돼요.

## 🙋 원장님이 자주 물어볼 것 (예진매니저 답변용)

**Q. 아이디가 뭐예요?**
→ 원장님 **휴대폰 번호**가 아이디예요. 비번은 처음에 `visual1234`.

**Q. 디자이너를 어떻게 넣어요?**
→ 「우리 지점 사람」 → 「바로 추가」로 이름·휴대폰 넣고 추가, 뜨는 아이디·비번을 그 친구에게 전달.

**Q. 아임웹/네이버는 매번 로그인해요?**
→ 아니요. **한 번만** 로그인해두면 그 브라우저에서 계속 유지돼요. 아임웹은 지점 공용, 네이버는 디자이너 각자 등록.

**Q. 그만둔 디자이너는요?**
→ 「우리 지점 사람」에서 그 사람을 **비활성화(내보내기)** 하면 로그인이 막혀요.

**Q. 우리 매출은 어디서 봐요?**
→ 「성과 대시보드」요. 「새로고침」을 누르면 어제 데이터까지 바로 가져와요.
