# 성과 대시보드 자동 수집 — 예진매니저 맥 설치 안내

HandSOS 성과 데이터는 **한국 가정용/매장용 인터넷 IP**에서만 수집돼요 (클라우드 서버는 HandSOS가 막음).
그래서 **예진매니저 맥**에서 수집을 돌립니다. 아래 3단계면 끝이에요. **처음 한 번만** 하면 됩니다.

## 준비물
- 예진매니저 맥 (한국 인터넷 연결)
- Node.js 설치 (안 돼 있으면 터미널에서 `brew install node`)
- 값 3개 (대표님/개발자에게 **안전하게** 전달받으세요 — 카톡·메일 말고 1Password 등):
  1. HandSOS 로그인 비밀번호
  2. Supabase URL
  3. Supabase 서비스 롤 키

## 설치 (한 번만)
1. 이 저장소를 맥에 받아요 (이미 있으면 최신으로 업데이트):
   ```
   git pull
   ```
2. **Finder에서** `visual-auto-web/scripts/setup-crawl-mac.command` 파일을 **더블클릭**해요.
   - "확인되지 않은 개발자" 경고가 뜨면: 우클릭 → **열기** → 열기.
3. 창이 뜨면 안내대로 값 3개를 붙여넣어요. 끝나면 자동으로 설치되고 첫 수집까지 해줘요.

## 설치되면 생기는 것
- **매일 08:30 자동 수집** — 맥이 켜져 있으면 알아서 돌아요. (꺼져 있었으면 다음에 켜질 때 최근 3일치를 자동 보정)
- **바탕화면 "지금 수집" 버튼** — 지금 당장 최신 데이터가 필요하면 더블클릭. 1~3분 뒤 앱 성과 대시보드에 반영돼요.

## 확인
앱 → **성과 대시보드** → **[주간]** 에서 지점·디자이너별 숫자가 보이면 성공.
수집 기록은 `~/.visualsalon/crawl.log` 에서 볼 수 있어요.

## 끄기
`visual-auto-web/scripts/uninstall-crawl-mac.command` 더블클릭 → 자동 수집과 버튼이 제거돼요.

## 주의
- 이 맥이 **아침 8:30쯤 켜져 있어야** 자동 수집이 돌아요. 늘 켜두거나, 안 되면 하루 한 번 "지금 수집" 버튼을 눌러주세요.
- 맥을 바꾸면 새 맥에서 설치 파일을 다시 한 번 실행하면 돼요.
