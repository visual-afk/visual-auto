## 📦 자동 발행을 위한 기술 인프라

### 1. 코드 + 자동화 인프라

| 항목 | 역할 | 상태 |
|------|------|------|
| **GitHub 저장소** | 코드 보관 + 자동 실행 (`visual-afk/visual-auto`) | 운영 중 |
| **GitHub Actions** | 월/수/금 자정 자동 실행 (cron 스케줄) | 운영 중 |
| **Gemini API (Google)** | 글 생성 + 이미지 생성 (Pro/Flash 자동 fallback) | 결제 등록 완료 |
| **Google Service Account** | 시트/캘린더/드라이브 자동 조작 권한 | 운영 중 |

### 2. Google Workspace 연동

| 항목 | 역할 |
|------|------|
| **구글시트** | 콘텐츠 일정 + 키워드 + 발행 URL 관리 |
| **구글캘린더** | 발행 일정 시각화 + 자동 동기화 |
| **구글독스 + 드라이브** | AI 생성 글 저장소 |

### 3. AI 지식베이스 (브랜드 SSOT)

| 뇌 | 내용 | 현재 완성도 |
|----|------|---------|
| **뇌1 SEO** | 키워드 전략, 네이버 SEO 규칙 | 일부 완성 |
| **뇌2 미용지식** | 31개 Index (브랜드/시술/원장 노하우/리뷰/오해 등) | 80% |
| **뇌3 소비자심리** | 페르소나, 감정 분류, 커뮤니티 데이터 | 80% |

## 📋 본부장님 확인/승인 필요한 항목

### 🔴 우선 확인 (운영 직결)

#### 1. API 비용 승인
- 현재 Gemini API 결제 카드 연결 완료
- 월 예상 비용: 글 30건 기준 **약 $15~20/월** (약 2만~3만원)
- 이미지 생성 추가 시 약간 더 발생

#### 2. 콘텐츠 발행 권한
- 현재 흐름: 자동 생성된 글 → 예진매니저 검토 → 아임웹/블로그 직접 발행
- **선택지 A**: 본부장님 사전 검토 단계 추가
- **선택지 B**: 검토 없이 예진매니저 재량으로 발행

→ 어느 방식으로 운영할지 결정 부탁드립니다.

#### 3. 브랜드 톤앤매너 검토
- 현재 적용된 뇌2/뇌3 (지식베이스) 검토 필요
- 특히 다음 메시지의 브랜드 일관성 확인 필요:
  - 결마지 차별점 ("코팅이 아니라 내부 구조를 채우는 시술")
  - 트리필드 펌제 (대표원장 자체 개발) 어필
  - "비주얼살롱은 정액권 강요하지 않는다" 메시지
  - "안 됩니다 거절 안 하고 대안 제시" 응대 철학

### 🟡 정기 확인 (장기 운영)

#### 4. 콘텐츠 일정 검토
- 현재 7월 말까지 콘텐츠 일정 짜둠
- 매월 다음 달 주제/일정 사전 승인 받을지 결정 필요
- 8월부터 어떻게 운영할지 방향성 결정

#### 5. 원장님 협조 요청
- **인영 원장님(강남신사점)** 답변 아직 못 받음
  - 질문 3가지:
    1. 결마지 가장 어려웠던 케이스
    2. 망쳐온 머리 복구 시 첫 확인 사항
    3. 약제 조합 판단 기준
- 다른 원장님들도 분기별 1회 업데이트 필요

#### 6. 사진/리뷰 권한
- 현재 비포애프터 드라이브 + 고객 리뷰 캡처 폴더 연결됨
- 블로그 글에 자동 삽입 가능한지 권한 확인 필요
- 초상권/저작권 이슈 본부장님 판단 필요
