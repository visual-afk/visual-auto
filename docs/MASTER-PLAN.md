# 비주얼살롱 블로그 완전 자동화 계획

## 한눈에 보기

```
예진매니저가 할 일          AI가 할 일                결과물
─────────────────    ──────────────────    ───────────────
월초: 시트에 주제 등록  →                        → 구글 캘린더에 일정 자동 등록
                     → 뇌1: SEO 분석
                     → 뇌2: 전문가 글쓰기        → 구글독스에 완성글 (캘린더에서 링크 확인)
                     → 뇌3: 소비자심리 최적화
사진 추가 + 검토      →                        → 아임웹/네이버 발행
                     →                        → 조회수/전환율 자동 추적
```

---

## 어떻게 돌아가는지

### STEP 1. 기획 (예진매니저)
매월 초에 구글시트에 이번 달 블로그 주제를 등록합니다.

| 월 | 주차 | 주제 | 키워드 | 글유형 | 상태 | 예정일 |
|---|-----|------|--------|--------|------|--------|
| 2026-04 | 2 | 봄철 두피케어 가이드 | 두피케어, 봄 헤어관리 | 정보형 | planned | 2026-04-14 |
| 2026-04 | 2 | 웨이브펌 비포애프터 | 웨이브펌, 디지털펌 | 스토리형 | planned | 2026-04-16 |
| 2026-04 | 3 | 여름 앞두고 염색 트렌드 | 여름염색, 하이라이트 | 시즌형 | planned | 2026-04-21 |

### STEP 2. 자동 동기화 → 캘린더
매일 아침 9시에 시트 내용이 구글 캘린더에 자동 반영됩니다.
- 캘린더에서 `[블로그] 봄철 두피케어 가이드` 이벤트가 보임
- 이벤트를 클릭하면 키워드, 글유형, 구글독스 링크가 설명에 있음

### STEP 3. AI가 글 작성 (월/수/금 10시 자동)
예정일이 오늘인 주제에 대해 AI 에이전트 3개가 순차적으로 글을 씁니다.

#### 뇌1: SEO 전략가
- 주어진 키워드의 검색의도 파악
- 네이버 SEO에 최적화된 제목/구조/태그 설계
- 참고 지식: `knowledge/seo/` 폴더

#### 뇌2: 미용전문가 + 브랜드 작가
- 비주얼살롱의 브랜드 보이스로 글 작성
- 서비스 전문 지식, 시술 노하우 반영
- 사진 넣을 위치 표시 ([IMAGE: 시술 전 모습] 등)
- 참고 지식: `knowledge/brand/` 폴더

#### 뇌3: 소비자심리 최적화
- 고객 공감 포인트 강화
- CTA(예약 유도) 자연스럽게 배치
- 제목을 클릭하고 싶게 최적화
- 참고 지식: `knowledge/consumer/` 폴더

### STEP 4. 구글독스에 완성글
AI가 글을 다 쓰면:
- 구글독스에 자동으로 문서가 생성됨
- 시트의 상태가 `draft_ready`로 바뀜
- 캘린더 이벤트 설명에 독스 링크가 추가됨
- **캘린더에서 해당 날짜 클릭 → 독스 링크로 바로 이동 가능**

### STEP 5. 검토 + 발행 (예진매니저)
1. 캘린더에서 오늘 날짜의 블로그 이벤트 클릭
2. 독스 링크를 눌러 글 확인
3. [IMAGE: ...] 표시된 곳에 사진 추가
4. 내용 검토 후 아임웹 + 네이버 블로그에 발행
5. 시트에 발행 URL 기록, 상태를 `published`로 변경

### STEP 6. 성과 추적 (자동)
매주 일요일에 발행된 글의 조회수/전환수를 자동 수집합니다.
(추적 방식은 추후 확정)

## 뇌(SSOT) 지식베이스

AI가 좋은 글을 쓰려면 비주얼살롱에 대해 잘 알아야 합니다.
`knowledge/` 폴더에 있는 파일들이 AI의 뇌입니다.

### 뇌1: SEO 전략 — `knowledge/seo/`

| 파일 | 뭐가 들어가야 하는지 |
|------|-------------------|
| `naver-seo-rules.md` | 네이버 블로그에서 상위 노출되려면 어떻게 써야 하는지 |
| `keyword-master.md` | 우리 살롱과 관련된 검색어 전체 목록 |
| `keyword-seasonal.md` | 봄/여름/가을/겨울별로 사람들이 많이 검색하는 미용 키워드 |
| `search-intent.md` | 키워드별로 사람들이 뭘 알고 싶어서 검색하는지 |
| `competitor-gaps.md` | 경쟁 살롱 블로그에서 안 다루는데 우리가 노릴 수 있는 주제 |

### 뇌2: 미용전문성 + 브랜딩 — `knowledge/brand/`

| 파일 | 뭐가 들어가야 하는지 |
|------|-------------------|
| `salon-identity.md` | 비주얼살롱은 어떤 곳인지 (위치, 분위기, 차별점) |
| `brand-voice.md` | 블로그 글의 말투와 분위기 (전문적? 친근한?) |
| `services-menu.md` | 전 메뉴 목록 + 가격 + 시술과정 + 걸리는 시간 |
| `staff-expertise.md` | 디자이너별 전문 분야, 경력, 자격증 |
| `product-lines.md` | 쓰는 제품 브랜드, 왜 그 제품을 쓰는지 |
| `case-studies.md` | 고객 시술 사례 (비포/애프터, 어떤 고민이었는지) |

### 뇌3: 소비자 심리 — `knowledge/consumer/`

| 파일 | 뭐가 들어가야 하는지 |
|------|-------------------|
| `personas.md` | 우리 주요 고객층은 누구인지 (나이, 직업, 성격) |
| `pain-points.md` | 고객들이 미용실에 대해 불안해하는 것들 |
| `desire-triggers.md` | 고객이 "예약해야지!" 하게 만드는 감정 |
| `decision-journey.md` | 고객이 미용실을 선택하는 과정 (검색 → 비교 → 예약) |
| `cta-patterns.md` | 글에서 예약을 유도하는 문구 패턴 |

## 예진매니저가 할 일 순서

### Phase A: 세팅 마무리 (1일)
→ `docs/SETUP-GUIDE.md` 보고 따라하기
- 구글 시트/캘린더/Drive에 서비스 계정 공유
- Google Cloud API 활성화
- 시트 헤더 세팅
- 테스트 실행

### Phase B: 뇌 구축 (3-5일)
→ `docs/YEJIN-GUIDE.md` 보고 Claude Code와 대화하면서 채우기
- 우선순위: brand/ → seo/ → consumer/
- Claude Code에게 "브랜드 보이스 파일 채워줘" 하고 대화하면 됨
- AI가 초안 쓸 수 있는 파일은 AI가 먼저 써줌

### Phase C: 운영 시작 (매월)
1. 매월 초: 시트에 주제 등록
2. 자동화가 돌아감 (캘린더에서 확인)
3. 독스에서 글 검토 + 사진 추가 + 발행
