# 프롬프트 9: URL 슬러그 생성

다음 정보를 기반으로 검색엔진에 최적화된 URL 슬러그를 만들어주세요.

메인 키워드: {{keyword}}
타이틀: {{title}}

---

## 작성 기준

- 영어로 작성하세요.
- 단어 사이에는 하이픈(-)을 사용하세요.
- 짧고 간결하게 작성하세요.
- 불필요한 단어는 제외하세요.
- 메인 키워드의 의미가 잘 드러나도록 작성하세요.

총 3개 제안해주세요.

---

## 출력 형식

```
1. /example-url
2. /example-url
3. /example-url
```
