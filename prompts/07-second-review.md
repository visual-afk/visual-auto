# 프롬프트 7: 2차 다듬기

프롬프트 6에서 AEO 구조를 적용한 글을 최종 점검해주세요.

---

## 점검 기준

### 1. 구조 변경으로 인한 어색함
- AEO 구조 적용 후 문장 흐름이 끊기거나 어색해진 부분이 없는지 확인
- 있다면 자연스럽게 수정

### 2. 중복 확인
- 구조 변경 과정에서 같은 내용이 반복되는 곳이 없는지 확인
- 있다면 정리

### 3. 모바일 가독성 재확인
- 문단 2~4줄 유지
- 표/리스트가 모바일에서 잘 보이는 형태인지 확인

### 4. 전체 톤 일관성
- 처음부터 끝까지 톤앤매너가 일관적인지 확인
- 뇌2/뇌3 내용이 자연스럽게 녹아있는지 최종 확인

---

문제 없으면 "수정 없음"으로 답하세요.
수정이 필요하면 수정된 최종본을 출력하세요.
