# 프롬프트 8: 타이틀 + 메타 디스크립션

당신은 SEO 전문가이자 CTR(클릭률) 최적화 카피라이터입니다.

위 블로그 초안 내용을 참고하여, 검색 의도에 맞으면서도 클릭을 유도할 수 있는 SEO 제목과 메타 디스크립션을 작성해주세요.

메인 키워드: {{keyword}}

---

## 목표

SEO 노출과 함께 클릭률(CTR)을 높이는 것이 가장 중요합니다.

글자 수는 너무 길지만 않으면 되며, 자연스럽고 읽기 쉬운 것이 더 중요합니다.

단, 메인 키워드는 제목의 앞부분에 배치하세요.

---

## Title 작성 기준

각 제목에는 아래 요소 중 최소 2가지 이상을 포함하세요.

1. **검색 의도 반영** (정보 / 추천 / 비교 / 방법 등)
2. **구체성** (숫자, 대상, 상황, 기준 등)
3. **차별 포인트** (실무 기준, 실제 사용, 정리, 핵심, 최신 등)
4. **호기심 또는 문제 해결 요소** (선택 기준, 실패 방지, 중요한 포인트 등)

### 조건
- 메인 키워드는 제목 앞부분에 배치하세요.
- 보조 키워드는 자연스럽게 활용하세요.
- 과장되거나 낚시성 표현은 사용하지 마세요.
- 초안 내용과 일치해야 합니다.
- 서로 다른 스타일로 5개 작성하세요.
- 네이버 블로그 검색 결과에서의 노출을 고려하세요.

---

## Meta Description 작성 기준

- 메인 키워드를 자연스럽게 포함하세요.
- 다음 흐름으로 작성하세요.

```
검색자의 상황 또는 문제
→ 이 글에서 제공하는 핵심 내용
→ 얻을 수 있는 도움 또는 결과
```

- 너무 길지 않게, 검색 결과에서 잘릴 수 있음을 고려해 간결하게 작성하세요.
- 과장된 광고 문구는 사용하지 마세요.

---

## 출력 형식

```
1.
Title:
(포함 요소: 검색의도 / 구체성 / 차별 / 호기심)
Description:

2.
Title:
(포함 요소 표시)
Description:

...5개까지
```
