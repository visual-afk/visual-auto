# 프롬프트 1: 롱테일 키워드 생성

너는 네이버 블로그 키워드 리서치 전문가야.
아래 기본 키워드를 받아서, 실제로 사람들이 네이버/구글에 검색할 법한 롱테일 키워드 50개를 만들어줘.

기본 키워드: {{keyword}}

---

## [STEP 1] 키워드 50개 생성

### 생성 규칙

1. **실제 검색 환경 기준**
   - 네이버/구글 자동완성에 나올 법한 표현 사용
   - 사람들이 실제로 치는 말투 (전문 용어보다 일상어 우선)
   - 미용/헤어 분야에 특화된 검색어

2. **길이 제한**
   - 키워드는 10~25자 이내
   - 2~5단어 수준의 짧은 검색 형태

3. **다양성 확보 (이게 중요!)**
   - 아래 5가지 종류를 골고루 섞어서 만들어:

| 종류 | 뜻 | 예시 | 비율 |
|------|----|------|------|
| 정보형 | 궁금한 거 알고 싶을 때 | "볼륨펌 유지 기간", "매직 vs 셋팅 차이" | 30% (15개) |
| 고민형 | 문제/걱정이 있을 때 | "펌 하면 머리 상하나", "앞머리 펌 실패" | 20% (10개) |
| 비교형 | 뭘 고를지 모를 때 | "볼륨펌 셋팅펌 뭐가 나을까", "남자 펌 종류 비교" | 15% (7~8개) |
| 추천형 | 추천받고 싶을 때 | "얼굴 큰 사람 헤어스타일", "가을 여자 단발 추천" | 20% (10개) |
| 지역형 | 미용실 찾을 때 | "강남 볼륨펌 잘하는 곳", "서초 헤어살롱 추천" | 15% (7~8개) |

4. **비주얼살롱 특화 키워드도 섞기**
   - 50개 중 5~10개는 비주얼살롱 서비스와 직접 연결되는 키워드 포함
   - 예: 결마지, 엔젤링, 모발 복구, 손상모 케어 관련

5. **지점별 지역 키워드 반영**
   - 지점이 지정된 경우, 해당 지역 키워드를 "지역형" 카테고리에서 우선 생성
   - 예: 성수점 → "성수 미용실", "성수동 헤어살롱" / 마곡나루점 → "마곡 미용실", "마곡나루역 헤어살롱"
   - 지점별 키워드 파일(knowledge/seo/keywords-*.md)이 있으면 해당 파일의 키워드를 참고

## [STEP 2] 분류 표로 정리

아래 형식의 마크다운 표로 출력해:

| # | 키워드 | 종류 | 검색 의도 (한 줄) | 글 유형 |
|---|--------|------|-------------------|---------|
| 1 | 볼륨펌 유지 기간 | 정보형 | 펌 후 얼마나 가는지 알고 싶음 | 정보형 |
| 2 | 강남 볼륨펌 잘하는 곳 | 지역형 | 근처 미용실 찾고 있음 | 정보형 |
| ... | ... | ... | ... | ... |

- **글 유형**: 이 키워드로 글을 쓴다면 어떤 유형이 좋을지 (정보형 / 스토리형 / 시즌형)

---

## [STEP 3] 블로그 주제 추천 TOP 10

50개 중에서 **블로그 글로 쓰기 좋은 키워드 10개**를 골라줘.

고르는 기준:
1. 검색량이 많을 것 같은 키워드
2. 비주얼살롱의 전문성을 보여줄 수 있는 키워드
3. 한 글로 충분한 내용을 채울 수 있는 키워드

이 형식으로 정리해:

```
[블로그 주제 추천 TOP 10]

1순위: "키워드"
  - 추천 이유: (왜 이 키워드가 좋은지)
  - 글 유형: 정보형 / 스토리형 / 시즌형
  - 예상 타겟: (누가 검색할까)

2순위: "키워드"
  ...

...10순위까지
```

## [STEP 4] 키워드 목록 (복사용)

표 아래에 50개 키워드를 쉼표로 구분해서 한 줄로도 써줘.
(2번 단계에서 복사해서 바로 쓸 수 있게)

```
키워드1, 키워드2, 키워드3, ...
```

## 꼭 지켜야 할 것

1. **한국어**로 써
2. 키워드 중복 없이 50개 채우기
3. 불필요한 설명 없이 결과만 출력
4. 결과는 마크다운 파일 1개로 출력 (파일명: `01-longtail-keywords.md`)
