# 예진매니저 검토 체크리스트

AI 9단계 파이프라인으로 생성된 글을 발행 전에 확인할 항목:

## 내용 확인
- [ ] 사실과 다른 내용이 없는가? (시술 방법, 효과 등)
- [ ] 비주얼살롱의 실제 서비스와 일치하는가?
- [ ] 가격이 언급되지 않았는가? (절대 금지)
- [ ] "손상 없이" 같은 금지 표현이 없는가?
- [ ] 과장 광고, 경쟁사 비하가 없는가?
- [ ] 브랜드 톤에 맞는가? (전문적이지만 친절)
- [ ] 이모지가 사용되지 않았는가?

## 구조 확인
- [ ] 각 H2 첫 문장이 결론/핵심 답변으로 시작하는가? (AEO)
- [ ] H2/H3 소제목이 검색 의도에 맞는가?
- [ ] 모바일에서 읽기 편한 구조인가? (문단 2~4줄)
- [ ] 표가 3열 이하인가? (모바일 깨짐 방지)

## 이미지
- [ ] [IMAGE] 표시된 곳에 적절한 사진 삽입했는가?
- [ ] 이미지 가이드(종류/구도/포인트)대로 찍었는가?
- [ ] 비포/애프터 사진이 필요한 곳에 전/후 나란히 추가했는가?
- [ ] 이미지 alt text가 작성되었는가?
- [ ] 최소 5장, 최대 10장 범위인가?

## SEO
- [ ] 타이틀: 키워드 앞배치, 25~40자인가?
- [ ] 메타 디스크립션: 80~150자, 키워드 포함인가?
- [ ] 태그: 5~10개인가?
- [ ] URL 슬러그: 영어 소문자 + 하이픈, 3~5단어인가?
- [ ] 메인 키워드가 첫 문단에 자연스럽게 들어갔는가?

## CTA
- [ ] CTA가 2~3회 자연스럽게 배치되었는가?
- [ ] 광고처럼 보이지 않는가?
- [ ] 압박형 CTA가 아닌가? ("지금 안 하면~" 같은 거 금지)
- [ ] 예약 유도로 자연스럽게 연결되는가?

## 발행
- [ ] 아임웹에 발행
- [ ] 네이버 블로그에 발행
- [ ] 구글시트에 발행 URL 기록
- [ ] 시트 상태를 `published`로 변경
