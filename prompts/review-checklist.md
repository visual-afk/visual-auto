# 예진매니저 검토 체크리스트

AI가 생성한 글을 발행 전에 확인할 항목:

## 내용 확인
- [ ] 사실과 다른 내용이 없는가?
- [ ] 비주얼살롱의 실제 서비스/가격과 일치하는가?
- [ ] 브랜드 톤에 맞는가?

## 이미지
- [ ] [IMAGE: ...] 표시된 곳에 적절한 사진 삽입했는가?
- [ ] 사진 품질이 괜찮은가?
- [ ] 비포/애프터 사진이 필요한 곳에 추가했는가?

## SEO
- [ ] 제목이 적절한가?
- [ ] 태그가 적절한가?

## 발행
- [ ] 아임웹에 발행
- [ ] 네이버 블로그에 발행
- [ ] 구글시트에 발행 URL 기록
- [ ] 시트 상태를 `published`로 변경
