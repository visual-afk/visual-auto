# 예진매니저 검토 체크리스트

AI 9단계 파이프라인으로 생성된 글을 발행 전에 확인할 항목:

## 내용 확인
- [ ] 사실과 다른 내용이 없는가? (시술 방법, 가격, 효과 등)
- [ ] 비주얼살롱의 실제 서비스/가격과 일치하는가?
- [ ] 브랜드 톤에 맞는가?
- [ ] 뇌2(미용지식/브랜드) 내용이 잘 반영되었는가?
- [ ] 뇌3(소비자 심리) 내용이 자연스럽게 녹아있는가?

## 구조 확인
- [ ] AEO 구조가 자연스럽게 적용되었는가?
- [ ] H2/H3 소제목이 검색 의도에 맞는가?
- [ ] 모바일에서 읽기 편한 구조인가? (문단 2~4줄)

## 이미지
- [ ] [IMAGE: ...] 표시된 곳에 적절한 사진 삽입했는가?
- [ ] 이미지 가이드(프롬프트 6)에서 제안한 유형과 맞는가?
- [ ] 사진 품질이 괜찮은가?
- [ ] 비포/애프터 사진이 필요한 곳에 추가했는가?
- [ ] 이미지 alt text가 SEO에 맞게 작성되었는가?

## SEO
- [ ] 타이틀이 적절한가? (키워드 앞배치, 30자 이내)
- [ ] 메타 디스크립션이 간결하고 키워드를 포함하는가?
- [ ] 태그가 적절한가? (5~10개)
- [ ] URL 슬러그가 깔끔한가?

## CTA
- [ ] CTA가 2~3회 자연스럽게 배치되었는가?
- [ ] 광고처럼 보이지 않는가?
- [ ] 설정한 목적(문의/이동/구매 등)에 맞는가?

## 발행
- [ ] 아임웹에 발행
- [ ] 네이버 블로그에 발행
- [ ] 구글시트에 발행 URL 기록
- [ ] 시트 상태를 `published`로 변경
