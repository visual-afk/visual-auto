## 검수 기준

### 1. 자연스러운 표현
- 어색한 표현, 번역체, 기계적인 문장이 없는지 확인
- 자연스럽지 않은 부분이 있다면 수정 문장 제안

### 2. 메시지 전달력
- 작성자가 전달하려는 핵심 메시지나 관점이 명확한지 평가
- 부족한 내용이 있다면 보완 제안

### 3. 정보의 충분성 (검색 만족도)
- 독자가 이 글을 읽은 후, 같은 검색어로 다른 글을 다시 찾을 필요가 없을 정도로 충분한지 평가
- 부족한 정보가 있다면 추가 제안

### 4. 독창성
- 단순 정보 정리에 그치지 않는지 확인
- 차별화를 위해 추가하면 좋은 관점, 설명, 사례 제안

### 5. 팩트체크
- 본문에 포함된 정보(시술 방법, 성분, 효과, 가격대 등)가 정확한지 웹검색으로 확인
- 잘못된 정보가 있다면 지적하고 올바른 내용으로 수정 제안
- 출처가 불분명한 주장이 있다면 표시

### 6. 뇌2/뇌3 반영 확인
- 뇌2(미용지식/브랜드 컨셉)의 전문성이 본문에 충분히 녹아있는지 확인
- 뇌3(소비자 심리)가 글의 흐름과 CTA에 반영되었는지 확인
- 부족하다면 구체적으로 어디에 어떤 내용을 추가할지 제안
